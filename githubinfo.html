All About GitHub

Question/Term Answer
What is a version control system? Software tools that help software teams manage changes to source code over time
What is a git? Git is a version control system used for tracking changes in computer files
What is a GitHub? GitHub is a code hosting platform for version control and collaboration
What is the difference betewen Git and GitHub? Git is a version control system and GitHub is a web-based hosting service
Who started Github and how was it started? GitHub was started by Chris Wanstrath, P.J. Hyett, Tom Preston0Werner and Scott Chacon using Ruby on Rails
What company owns it now? The current owner of GitHub is Microsoft Corp.
How much does a GitHub account cost? Accounts are free for standard users but range from 3 to 20 dollars depending on the usage needs
What is the Octocat? The Octocat is the mascot of GitHub
Repository A place where you can store your code, your files, and each file's revision history
Commit A commit records changes to one or more files in your branch
Fork A fork is a new repository that shares code and visibility settings with the original upstream repository
Push Push is used to upload local repository content to a remote repository
Pull Request Pull requests let you tell others about changes you've pushed to a branch in a repository on GitHub
Workflows A workflow is a configurable automated process that will run one or more jobs
Issues Issues let you track your work on GitHub, where development happens
Raw button With the raw button, you can view or copy the raw content of a file without any styling
Blame button This view gives you a line-by-line revision history, with the code in a file separated by commit
