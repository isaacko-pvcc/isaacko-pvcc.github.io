Programming Languages and Terminology
Term Definition
Machine Languagea The numeric codes for the operations that a particular computer can execute directly.
Debugging The process of finding and fixing errors or bugs in the source code of any software.
Assembler Software that converts an assembly language code to machine code
Assembly Language Any low-level programming language with a very strong correspondence between the instructions in the language and the architecture's machine code instructions.
Translator Program that converts source code into object code.
Compiler A computer program that translates computer code written in one programming language (the source language) into another language (the target language).
Interpreter An interpreter is a computer program that directly executes instructions written in a programming or scripting language.
Cross-platform Software Cross-platform software is computer software that is designed to work in several computing platforms.
Natural language Natural language processing (NLP) refers to the branch of computer science—and more specifically, the branch of artificial intelligence or AI—concerned with giving computers the ability to understand text and spoken words in much the same way human beings can.
Formal language A formal language consists of words whose letters are taken from an alphabet and are well-formed according to a specific set of rules called a formal grammar.
Machine independence A term applied to software that is not dependent on the properties of a particular machine.
High-level programming language Provide strong abstraction from the details of the computer and their main advantage is they are portable and can run on different types of computer hardware.
Object-oriented programming Object-oriented programming (OOP) is a programming paradigm based on the concept of objects, which can contain data and code.
Scripting language A programming language that employs a high-level construct to interpret and execute one command at a time.
Integer A datum of integral data type, a data type that represents some range of mathematical integers.
Float Floating-point variables are represented by a mantissa, which contains the value of the number, and an exponent, which contains the order of magnitude of the number.
String A string is made up of characters, and can include letters, words, phrases, or symbols.
Primitive data types Primitive data types are a set of basic data types from which all other data types are constructed.
Data structure A data structure is a data organization, management, and storage format that is usually chosen for efficient access to data.
List Lists are used to store multiple items in a single variable.
Tuple A type of data structure that is very similar to lists.
Set A data type in python used to store several items in a single variable.
Dictionary A kind of data structure that stores items in key-value pairs.
Function A function is a block of code which only runs when it is called.
Local variable Local variables in python are those variables that are declared inside the function.
Global variable A global variable in Python is often declared as the top of the program. In other words, variables that are declared outside of a function are known as global variables.
Source Program Source code is what we call the human-readable computer instructions written by programmers.
Object Program A programming paradigm that provides a means of structuring programs so that properties and behaviors are bundled into individual objects.
